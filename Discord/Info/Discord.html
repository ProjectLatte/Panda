We use Discord to chat about several things. If you don't know what Discord is then you can click on the "What is Discord" link below.
If you want to become a Let's Explorer then it's probably best to join our Discord server, where we can discuss it with you. Do you have any questions to ask Let's Explorers? Well, all LEs are there! So, check it out below. These are all the channels on the Discord right now - #announcements - For all announcements related to the Let's Explore community #darules - Read this channel first. It contains all of the rules. #general - Talk about anything related to Let's Explore! #notability - You'll be able to access this chat after you get the "Notable Subjects" role - you get it if you're a regular user of the Discord server. #victortranmemescentral - Post all of your Victor Tran memes! #challenges - Where some occasional challenges will take place. #councilofexplorers - If you become a Let's Explorer, you can chat about anything there! #random - For all of the off-topic moments. #new-videos - This is where you post your new videos if you're a Let's Explorer, or if you want to see other's LE channels, go there! #punishmenthistory - This is where all the bad stuff people have done go... #bot - Use our nice bots there! #projectlatte - Discuss anything about Project Latte!
